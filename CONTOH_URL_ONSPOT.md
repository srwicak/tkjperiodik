## ğŸ†˜ Troubleshooting

### Error: "Pendaftaran langsung tidak tersedia"
**Solusi:**
1. Cek checkbox "Izinkan Pendaftaran Langsung" sudah dicentang
2. Pastikan status ujian: Aktif
3. Pastikan ada jadwal ujian hari ini

### Error: "Ujian tidak ditemukan"
**Solusi:**
1. Cek slug ujian benar
2. Ujian mungkin sudah dihapus
3. Ujian masih berstatus Draft

### Form Tidak Bisa Diisi
**Solusi:**
1. Refresh browser (Ctrl+F5 atau Cmd+Shift+R)
2. Clear cache browser
3. Coba browser lain
4. Cek koneksi internet

### URL Tidak Bisa Dibuka
**Solusi:**
1. Cek typo di URL
2. Pastikan tidak ada spasi di URL
3. Pastikan format: `/pendaftaran-langsung/slug` (dengan dash/minus)
4. Jangan ada karakter khusus selain dash
